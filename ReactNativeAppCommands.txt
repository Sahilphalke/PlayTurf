//Backend
Install Node Modules : npm Install
Install bcrypt : npm install bcrypt   
install Prisma@5 : npm install prisma@5 @prisma/client@5    
install nodemon : npm install nodemon --save-dev
install express cors dotenv pg : npm install express cors dotenv pg   

//frontend App
Create React Native App : npx create-expo-app mobile
Reset React Native App : npm run Reset-project
To run App : npx expo
Nativewind : npm install nativewind
Initialize Tailwind : npx tailwindcss init
